Accessible Button

This button achieves accessibility through it's high contrast label, selection colors for tab navigations and ...

HTML

            <button class="accessible-button">Submit</button>

CSS

            .accessible-button {
  background-color: #8C1E41;
  color: #FFF;
  padding: 10px;
  cursor: pointer;
  box-shadow: black 2px 2px;
}

.accessible-button:hover,
.accessible-button::selection {
  box-shadow: black 3px 3px;
}

.accessible-button:active {
  box-shadow: none;
}

Accessible Toggle Switch

This button achieves accessibility through it's high contrast label, selection colors for tab navigations and ...

HTML

            <label class="switch">
  <input type="checkbox">
  <span class="slider round"</span>
</button>

CSS

            /*Accessible Toggle Switch via W3 Schools*/
/* The switch - the box around the slider */
.switch {
  position: relative;
  display: inline-block;
  width: 60px;
  height: 34px;
}

/* Hide default HTML checkbox */
.switch input {
  opacity: 0;
  width: 0;
  height: 0;
}

/* The slider */
.slider {
  position: absolute;
  cursor: pointer;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background-color: #ccc;
  -webkit-transition: .4s;
  transition: .4s;
}

.slider:before {
  position: absolute;
  content: "";
  height: 26px;
  width: 26px;
  left: 4px;
  bottom: 4px;
  background-color: white;
  -webkit-transition: .4s;
  transition: .4s;
}

input:checked + .slider {
  background-color: #2196F3;
}

input:focus + .slider {
  box-shadow: 0 0 1px #2196F3;
}

input:checked + .slider:before {
  -webkit-transform: translateX(26px);
  -ms-transform: translateX(26px);
  transform: translateX(26px);
}

/* Rounded sliders */
.slider.round {
  border-radius: 34px;
}

.slider.round:before {
  border-radius: 50%;
}
